# 🎓 Placement Prediction using Machine Learning

A machine learning project that predicts whether a student is likely to get placed based on their academic profile, technical skills, internships, projects, and other relevant attributes.

---

## 📌 Project Overview

This project uses a supervised machine learning classification model to predict student placement status. It includes data preprocessing, feature engineering, model training, and evaluation.

---

## 🚀 Features

* Data preprocessing
* Label Encoding for categorical features
* Feature Engineering
* Train-Test Split
* XGBoost Classifier
* Model Evaluation using:

  * Accuracy Score
  * Classification Report
  * Confusion Matrix

---

## 📂 Dataset Features

The dataset contains information such as:

* Gender
* Degree
* Branch
* CGPA
* Projects
* Internships
* Aptitude Score
* Coding Skills
* Communication Skills
* Placement Status (Target)

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* XGBoost

---

## 📊 Project Workflow

1. Load the dataset
2. Clean and preprocess the data
3. Perform feature engineering
4. Encode categorical variables
5. Split the data into training and testing sets
6. Train the XGBoost classifier
7. Evaluate the model using standard classification metrics

---

## 📈 Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Precision
* Recall
* F1-Score
